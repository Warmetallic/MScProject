The Greig Duncan Folk song Collection Website User manual

1Introduction
1.1Purpose of writing
The purpose of this user manual is to help users understand the this folk song website and learn how to operate the system. The readers of this user manual are: music folklore scholars who use the system, system maintainers, teachers and students who use the system. 1.2Project background
The Greig Duncan Folk song Collection Online website project was completed by the team alpha which is consisted of Msc IT postgraduates. All the data was provided by Dr.Thomas who is also the client of this project. 2.Software Overview
2.1Functional Overview
This system is used to realize the storage and display of some music resources. This system provides users with inquiries about music, browsing of music scores, playing of video and audio, and adding, deleting, and updating other information. Data backup and restore, user password management and other functions. Enable users to make better use of the folk music resources provided by the school to complete in-depth research. 2.2Operating environment 2.21Hardware
The server supports PHP, PostgreSQL services, and web services. 2.22Support software and system
Support system: Windows, Unix and Mac OS. Browser: Support IE8 to IE11 version, Google and other browsers using IE core or IE compatibility mode. 3.Instructions for use
●User register: The way to access this website, you need to open the browser, enter the system address “https://stark-mesa-89039.herokuapp.com/”.
●Admin register: The quick link for dashboard to manage is: “https://stark-mesa-89039.herokuapp.com/admin/login/?next=/admin/”.
●Browse web content: Users can select the content they want to browse by sliding the scroll bar on the far right.
●Access to admin page: System maintainers can access the admin page by click the “admin” button.
User operation examples:
●Search songs: Users can search aimed songs by using advanced search engine. Songs name, authors name, lyric and genre all can be the key words content to help users do some search work. Users choose to use filters would have a higher effective result.Just put the volumes name in the first column, and put the author name in the second column that will increase the efficiency. After complete to fill the key words, what you need is to press the “search” button or “enter” on your keyboard.

Task-oriented Guide


  1. How to add a new song?

    • Click on the ADMIN link
    • Enter the authorisation details (Login)
    • Click on Songs
    • Choose to ADD SONG
    • Fill all fields
    • Press save
    • Refresh the website. Your song is now diplayed on the home page!

  2. How to search a song?

    • Use the navigation bar on the top of the website
    • Enter a song's title in the Search Songs field
    • Click the Search button
    • For the advance search click on the Filters button
    • Fill the fields with additional information for a more precise result
    • Click the Search button
